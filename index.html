Examples

Load the list with data frames with beta chain data and V- and J-genes alphabets for human subjects:

data(twb)
data(human.alphabets)

Get a number of clones, in-frames and out-of-frames, number of unique amino acid CDR3 sequences and other basic statistics:

mitcr.stats(twb)

Compute V-usage of the first subject:

freq.Vb(twb[[1]])

Compute V-usage of all subjects and plot a histogram:

vis.V.usage(twb, .dodge = T)

Compute a pairwise number of shared clones and plot a heatmap:

twb.int <- intersect(twb, 'n0e')
vis.heatmap(twb.int)
